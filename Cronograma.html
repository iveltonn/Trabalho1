Cronograma

Data Descrição
Dia 06/05 Neste dia, serão abordados os conceitos básicos do desenvolvimento web, incluindo linguagens de marcação, estilos e interação cliente-servidor.
Dia 07/05 Introdução ao HTML: Será uma introdução à linguagem de marcação HTML, cobrindo sua estrutura básica, tags comuns e semântica.
Dia 08/05 Utilizando CSS para estilizar o HTML: Este dia será dedicado ao uso de CSS para estilizar páginas HTML, incluindo seletores, propriedades e técnicas de layout.
Dia 09/05 Como utilizar o GitHub: Será uma introdução ao uso do GitHub para controle de versão de código, colaboração e hospedagem de projetos.
Dia 10/05 Implementação de Desenvolvimento Web na Engenharia Agronômica: Neste último dia, será discutido como aplicar os conceitos de Desenvolvimento Web na engenharia agronômica incluindo caso de uso e exemplos práticos.
